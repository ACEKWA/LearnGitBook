2.3 安装Gitbook

安装十分简单，只需要一行命令即可。

npm install -g gitbook-cli

接下来，查看一下版本，如果结果有个版本号，说明安装成功。

gitbook -V

其实，第一个命令只是下载相应的版本，并没有真正安装。 所以在第一次查看版本的时候，会自动安装，安装完之后显示版本号。
